# Github Explorer

## Getting Started

First, run the development server:

```bash
yarn dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

## About the program

This repository is a program to find users and github repositories.

## Stack

1.  React
2.  Vite
3.  Octokit
4.  Sass
